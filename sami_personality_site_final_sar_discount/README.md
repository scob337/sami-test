# موقع اعرف شخصيتك، نسخة الغلاف الجديد والأسعار بعد الخصم

## التحديثات
1. تم استبدال الصورة القديمة بالصورة الجديدة.
2. تم تحسين الغلاف وحفظه بصيغة PNG.
3. تم تجهيز نسخة HTML مدمجة بالصورة داخل الملف:
   - index-embedded-cover.html
   هذه النسخة تفتح مباشرة وتظهر الصورة بدون الحاجة لملفات خارجية.
4. تم تعديل الأسعار بعد الخصم:
   - الكتاب الرقمي + الاختبار: 36 ر.س بدل 99 ر.س
   - الكتاب + الاختبار + التقرير: 39 ر.س بدل 149 ر.س

## ملاحظات للمبرمج
- استخدم العملة SAR في بوابة الدفع.
- data-price للكتاب مع الاختبار = 36
- data-price للباقة الكاملة = 39
- لا يتم إطلاق حدث Purchase إلا بعد نجاح الدفع.
- لا تضغط صورة الغلاف ضغطًا قويًا عند رفعها للسيرفر.
